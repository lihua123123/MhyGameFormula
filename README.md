# 米家三游的伤害计算公式

可以直接去拿html文件打开，不过这个更新频率会慢

也可以直接访问网站使用

最后如果想本地保存最新版本的话按照以下步骤使用：

- 拉取该项目到本地

- 安装[node.js](https://nodejs.org/zh-cn)

- 运行以下代码：

    ```cmd
    npm install && REM 安装环境
    ```

    ```cmd
    npm run build-single && REM 生成html文件到/dist-single文件夹下
    
    npm run dev && REM 在线打开
    ```

然后就可以查看了
